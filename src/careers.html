Careers in Web and User Experience:

  • UI Designer: Focuses on designing the user interface of a website
  • UX Designer: Focuses on designing the user experience of a website
  • Front-End Developer: Adds visual elements and works on the overall website appearance
  • Information Architect: Hones the layout of a website or app to make it easy to locate things
  • User Experience Manager: Manages and directs UX teams, overseeing the entire process
